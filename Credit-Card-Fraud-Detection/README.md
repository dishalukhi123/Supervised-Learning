# 📌 About The Project

Credit card fraud is one of the major challenges in digital payment systems. Since fraudulent transactions account for only a very small percentage of all transactions, traditional machine learning models often struggle with class imbalance.

This project develops an end-to-end Fraud Detection System using Machine Learning techniques. The application predicts whether a transaction is legitimate or fraudulent and provides interactive visualizations through a Streamlit dashboard.

# 📂 Dataset

**Dataset:** Credit Card Fraud Detection

Source:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

### Dataset Information

- Total Transactions: **284,807**
- Fraud Transactions: **492**
- Features: **30**
- Target Variable:
  - 0 → Legitimate
  - 1 → Fraud
